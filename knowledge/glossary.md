# Glossary

*Last updated: 2026-07-11*

| Term | Meaning |
|------|---------|
| **Uma** | An umamusume / trainee. |
| **CM** | Champions Meeting — monthly PvP tournament on a fixed course; the main endgame. |
| **Team Trials** | Daily async PvP with a 15-uma team. |
| **Bashin** (身) | Horse-length margin unit ≈ 2.5m; umalator reports comparisons in mean/median bashin. |
| **Proc** | A skill activating in a race. |
| **Spurt / full spurt** | The final all-out run; "full spurt" = had enough HP to run it at max speed the whole way. |
| **Spurt rate** | % of simulated runs achieving a full spurt — the correct stamina-requirement metric. |
| **Rushed / kakari** | Losing control early in the race, burning extra HP. Reduced by wit. |
| **Nige / Senkou / Sashi / Oikomi** | JP terms for Front Runner / Pace Chaser / Late Surger / End Closer. |
| **Pacer** | The front-running uma that sets the race pace (customizable in umalator-global). |
| **Deck** | The 6 support cards taken into a career. |
| **MLB** | Max limit break — a support card at 4/4 copies merged. |
| **Rainbow** | Friendship training — big gains from a bonded card on its specialty facility. |
| **Sparks** | Inheritance factors (blue = stats, pink = aptitudes, green = unique skill, white = skills/races). |
| **Hint** | Skill discount level (Lv1–5 = 10/20/30/35/40% off) from support card events. |
| **◎ / ○** | Greater / lesser version of a passive skill. |
| **Debuffer** | An uma built to hinder opponents rather than win. |
| **URA** | The launch career scenario (URA Finale). |
| **GameTora** | Main community database site. |
